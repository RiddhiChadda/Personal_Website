# Personal Portfolio Website

This is my personal portfolio website to showcase my profile, skills, projects, and contact information. It is built using HTML, CSS, and JavaScript.

## Features
- Home page with profile, education, skills, and work experience
- Portfolio page showcasing projects
- Contact page with a simple form
- Responsive design
- Interactive elements like typing animation and copy-to-clipboard email

## Technologies Used
- HTML5
- CSS3
- JavaScript
